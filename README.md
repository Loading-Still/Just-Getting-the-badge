## Quickdraw Badge

To get this bages follow these steps.

- Creat a new repositorie.
- Make new issue on the repo.
- Then close the issue in less than five minutes.

you may have done that by accident, sense some time you spot an issue then it was easy to fix or you just thought it was any issue but it wasn't.

## Pull Shark 

Go in any repository that you have and edit any file or the main repo, but commit changes in a new branch.
- Creat a pull request (then give it some time)
- Merge the pull request ⇒ Confirm merge
- Do the same process ones more

Congratulates you got the badge.

## Yolo 
If you ever get into a collaboration with others on a project you will get this badge, but if you work alone most of the time you will have to make another account to get it.

- Open the repository then go to settings
- In access you will find collaborators
- Add people ⇒ search for the other account name ⇒ add to repository
- Switch to the other account and accept the invitation from the inbox
- Back to main account ⇒ edit any file 
- After making any changes commit changes (Create new branch option) ⇒ Propose changes 
- On the right there is reviewers add the other account ⇒ create pull request ⇒ then merge it

Congratulations, now you have 3 badges.
